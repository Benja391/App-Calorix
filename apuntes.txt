Apuntes proyecto Calorix - App Control de Calorías
1. Idea del proyecto
Aplicación para controlar calorías consumidas y gastadas durante el día, registrando comidas y entrenamientos, y mostrando el balance final.

2. Tecnologías usadas
Backend: Node.js + Express

Base de datos: MongoDB Atlas

Frontend: Vue 3 + Vite + Tailwind CSS

3. Configuración Backend
Crear carpeta calorix-backend
